## 项目简介
这是从原 Expo / React Native 项目迁移出的纯前端 Web App / PWA 版塞尔维亚Loto彩票选号应用，用户可以通过自选或机选方式选择7个号码（范围01-39）进行投注。

## 当前功能
- 自选 7 个号码，范围 01–39
- 机选 7 个不重复号码
- 清除当前选号
- 保存选号到历史记录
- 从历史记录重新使用号码
- 从历史记录出现过的号码中随机生成组合
- 删除单条历史记录
- 清空全部历史记录
- 号码统计
- 收藏号码
- 分享号码；不支持系统分享时会自动复制到剪贴板
- PWA：支持添加到手机主屏幕，支持基础离线缓存
